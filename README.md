# 3D Fitting Room

## Project Documentation
This project is a 3D fitting room application that allows users to try on clothes and accessories virtually. The application provides a realistic experience of how the items would look on the user.

## Setup Instructions
1. **Clone the repository:**  
   ```bash
   git clone https://github.com/kirazhilts-code/3d-fitting-room.git
   cd 3d-fitting-room
   ```
2. **Install dependencies:**  
   Make sure you have Node.js and npm installed, then run:
   ```bash
   npm install
   ```
3. **Run the application:**  
   ```bash
   npm start
   ```
   The application will be accessible at `http://localhost:3000`.

## Features
- **Virtual Try-On**: Users can select items and see how they fit without physically trying them on.
- **3D Modeling**: Realistic 3D representations of clothing.
- **User Interaction**: Users can rotate and zoom in/out on the items.
- **Easy Setup**: Simple instructions for setting up the project on local machines.
